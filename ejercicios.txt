## üìä Tabla Comparativa de Operaciones Comunes

| Operaci√≥n | Python | JavaScript | Dart |
|-----------|--------|------------|------|
| **Crear mapa/objeto** | `dict = {'a': 1}` | `obj = {a: 1}` | `Map<String, int> m = {'a': 1}` |
| **Agregar elemento** | `dict['b'] = 2` | `obj. b = 2` o `obj['b'] = 2` | `m['b'] = 2` |
| **Eliminar elemento** | `del dict['a']` o `dict.pop('a')` | `delete obj. a` | `m.remove('a')` |
| **Verificar clave** | `'a' in dict` | `'a' in obj` | `m.containsKey('a')` |
| **Obtener valores** | `dict.values()` | `Object.values(obj)` | `m.values` |
| **Iterar** | `for k, v in dict.items()` | `for (let [k, v] of Object.entries(obj))` | `m.forEach((k, v) => ...)` |
| **Filtrar lista** | `[x for x in list if x > 5]` | `list.filter(x => x > 5)` | `list.where((x) => x > 5)` |
| **Mapear lista** | `[x*2 for x in list]` | `list.map(x => x*2)` | `list.map((x) => x*2)` |
| **Reducir lista** | `from functools import reduce`<br>`reduce(lambda a,b: a+b, list)` | `list.reduce((a,b) => a+b)` | `list.reduce((a,b) => a+b)` |
| **Ordenar** | `sorted(list, reverse=True)` | `list.sort((a,b) => b-a)` | `list.sort((a,b) => b.compareTo(a))` |
